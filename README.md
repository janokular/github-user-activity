## GitHub User Activity
### Initial setup
```
# Create and activate the virtual environment
python3 -m venv .venv
. ./.venv/bin/activate

# Installation
pip3 install .

# Exit the virtual environment
deactivate

# Uninstallation
pip3 uninstall github_activity
```

### Running the application
```
# Fetch GitHub user activity
github_activity USERNAME
```
